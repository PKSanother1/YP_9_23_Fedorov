# YP_9_23_Fedorov
## Первая отправка в гит 
```
git clone - клонирование папки с репозитория 
git add - это добавление в изменения для коммита 
Чтобы сделать определенное добавление пишем git add название_файла
Если хотите все добавить из папки - git add .
git commit -m "И тут сообщение для коммита"
-m флажок нужен для сообщения для коммита

Чтобы отправить набираем 
git push
```
